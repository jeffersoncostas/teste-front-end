
Esse projeto foi criado utilizando o ReactJS

## Como rodar o projeto

É necessário clonar o projeto, abrir o terminal e utilizar os comandos:

 - `npm install` Para instalar as dependências.
 - `npm start` ou `yarn start`  Para inicializar  o projeto na máquina local.
 - Abrir  http://localhost:3000 para ver no browser.
